# Pavan's Files (Jut-Score Web)

A minimalist, high-performance web interface for managing private files stored on GitHub.

## Features
- **Minimalist UI:** No hero sections, no fluff. Just your files and folders.
- **Mobile First:** Fully responsive grid layout that works perfectly on phones.
- **GitHub Storage:** Uses GitHub repository as the backend for file storage.
- **Secure Access:** Protected by a simple password gate.
- **Fast Actions:** Upload, create folders, search, and delete with ease.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Create a `.env` file based on `.env.example`.
4. Run the app: `python app.py`.
